# Tokens Principales

Empoderando a Unit Network: La Esencia de los Tokens Principales Impulsando la Innovación y el Crecimiento.

Los Tokens Principales, meticulosamente creados por la DAO de Unit Network, forman la columna vertebral de nuestro ecosistema financiero.

Verifica su creación explorando la cuenta 'Manager' en la Visión general de tokens.

Cada Token Principal está dotado de funcionalidades y tokenómicas distintas, gestionadas de forma independiente por las DAOs del Ecosistema y los Contribuyentes.

Adéntrate en los fundamentos de cada tipo de token para navegar por este espacio emergente, equipándote para lanzar tu propio token con confianza.
