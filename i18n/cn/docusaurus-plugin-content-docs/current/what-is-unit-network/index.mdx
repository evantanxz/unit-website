# 什么是 Unit Network？

Unit Network：引领基于 Token 的新时代

Unit Network 开创了基于 Token 经济的新时代，既是一种协议又是一种应用。它连接社区、项目、企业、现实世界资产和网络状态，构建了一个开放的金融网络。

作为一个全球可访问的平台，Unit Network 以包容性为基础，提供点对点和可互操作的金融基础设施。

这个去中心化的协议旨在创建一个更加公平的金融系统，提供专为人、项目和企业的潜在增长而设计的工具和应用，拒绝歧视，并实现人、项目和企业普遍参与的目标。

Unit Network 超越了数字货币，允许用户将有形资产如音乐、房地产、基础设施、知识产权等进行代币化。

该平台促进了多样资产的安全表达、交易和数字化，开创了灵活性和创新的新时代。

在价值互联网领域，Unit Network 打破了向“免费”互联网服务交出控制权和个人数据访问的常规做法。

通过去中心化的应用程序和工具，用户拥有控制权，开设账户是免费的，容易操作，完全在用户的控制之下，而且不收集个人信息。这确保了在不损害隐私的情况下体验到价值互联网的体验。
