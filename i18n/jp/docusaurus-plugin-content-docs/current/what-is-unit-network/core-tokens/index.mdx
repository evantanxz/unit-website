# コアトークン

Unit Network を力強く支える: イノベーションと成長を推進するコアトークンの本質。

コアトークンは、Unit Network DAO によって綿密に作られ、私たちの金融エコシステムの骨格を形成しています。

トークンの概要で 'Manager' アカウントを探索することで、それらの作成を確認してください。

各コアトークンは異なる機能とトークノミクスを備えており、エコシステム DAO とコントリビュータによって独自に管理されています。

この新興の領域を探索し、自信を持って独自のトークンを立ち上げるために、各トークンタイプの基本を深く掘り下げてください。
