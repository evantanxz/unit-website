# ユニットネットワークとは？

ユニットネットワーク: トークンベースの新時代の先駆者

ユニットネットワークは、プロトコルとアプリケーションの両方として機能し、コミュニティ、プロジェクト、ビジネス、実世界の資産、およびネットワーク状態をオープンな金融ネットワークでつなぎます。

ユニットネットワークは、世界中でアクセス可能で、包括性に富み、ピア・ツー・ピアおよび相互運用可能な金融インフラを提供しています。

分散型のプロトコルは、より公正な金融システムを作り出すことを目指しており、差別を排除し、人々、プロジェクト、ビジネスの潜在的な成長への普遍的な参加を可能にするためのツールとアプリケーションを提供しています。

ユニットネットワークはデジタル通貨を超え、ユーザーが音楽、不動産、インフラ、知的財産などの有形資産をトークン化できるようにします。

プラットフォームは、さまざまな資産の安全な表現、取引、およびデジタル化を促進し、柔軟性とイノベーションの時代を迎えます。

バリューのインターネットの領域では、ユニットネットワークは「無料」のインターネットサービスのために個人データの制御とアクセスを放棄する通常の慣習を打破します。

分散型アプリケーションとツールがユーザーコントロールを優先し、アカウントの開設は無料で、簡単で、完全にユーザーのコントロール内で行われ、個人情報は収集されません。これにより、プライバシーを犠牲にすることなくバリューのインターネット体験が実現されます。
