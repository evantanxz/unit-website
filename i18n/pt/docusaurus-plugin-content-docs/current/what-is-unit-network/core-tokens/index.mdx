# Tokens Principais

Empoderando a Unit Network: A Essência dos Tokens Principais Impulsionando Inovação e Crescimento.

Os Tokens Principais, meticulosamente criados pelo DAO da Unit Network, formam a
espinha dorsal do nosso ecossistema financeiro.

Verifique sua criação explorando a conta 'Manager' na visão geral dos tokens.

Cada Token Principal é dotado de funcionalidade e tokenômica distintas, gerenciadas independentemente pelos DAOs do Ecossistema e Contribuidores.

Aprofunde-se nos fundamentos de cada tipo de token para navegar neste espaço emergente, equipando-se para lançar seu próprio token com confiança.
