# Tokens principaux

Autonomiser Unit Network : L'essence des Tokens principaux alimentant l'innovation et la croissance.

Les Tokens principaux, méticuleusement conçus par l'Unit Network DAO, forment l'épine dorsale de notre écosystème financier.

Vérifiez leur création en explorant le compte 'Manager' dans la section Aperçu des tokens.

Chaque Token principal est doté de fonctionnalités et de tokenomics distincts, gérés indépendamment par les DAO de l'écosystème et les contributeurs.

Plongez dans les fondamentaux de chaque type de token pour naviguer dans cet espace émergent, vous équipant pour lancer votre propre token en toute confiance.
